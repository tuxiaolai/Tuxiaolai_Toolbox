# 工具箱主窗口模块

统一的主窗口框架，以 Activity Bar + 内容区布局集成所有工具模块。

## 主要类

| 类 | 文件 | 说明 |
|----|------|------|
| `ToolboxWindow` | `ToolboxWindow.h/.cpp` | 主窗口类，继承 `QMainWindow`，管理侧边栏与活动面板 |

## 布局结构

```
[侧边栏 50px 固定] [活动栏 ⇔ 内容区]
     │                    │
     ├ 文件管理器图标    ├ 文件管理器 / 浏览器 (QStackedWidget)
     └ 浏览器图标       └ 预留内容区
```

- **侧边栏**：50px 固定宽度图标栏，可拖拽移动窗口
- **活动栏**：可拖拽调整宽度的侧面板，`QSplitter` 分割
- **内容区**：预留右侧空间，供未来工具模块扩展

## 交互

- 点击侧边栏图标切换活动模块
- 再次点击当前活动图标隐藏活动栏
- 侧边栏空白区域拖拽移动窗口
- 活动栏与内容区之间的分割条可拖拽调整宽度

## 样式

内置现代简约暗色主题 QSS：
- 深邃底色 `#0d0d0f`
- 柔和对比 + 蓝色强调 `#5b9aff`
- 支持右键菜单、ToolTip 统一样式

## 入口

```bash
cmake --build build --target Tuxiaolai_Toolbox
```
