# 2026-06-20

## feat/toolbox-file-try1 — Toolbox 主窗口集成

### 布局演进
经历了多轮布局重构才最终确定：

1. **第一版**：QSplitter 左侧折叠面板 + 内容区（VS Code 风格 CollapsiblePanel）
   - 折叠到 36px 时触发 Windows geometry 警告
   - 修复为 64px + 按钮放大

2. **第二版**：Activity Bar (50px) + QSplitter(Side Panel | Content)
   - 活动栏拖拽移动窗口
   - 侧栏 200px 固定不可拖拽（在 QSplitter 内）
   - 侧栏高度不与窗口同步变化

3. **第三版**：去掉了 QSplitter，侧栏直接 50px 固定
   - 文件管理器功能丢失

4. **最终版**：[侧边栏 50px 固定] [活动栏 ⇔ 内容区]
   - 侧边栏: 50px 固定, 拖拽移动窗口, 点击图标切换活动栏
   - 活动栏: 200px, 含完整文件管理器, QSplitter 可拖拽调整
   - QLabel 代替 QPushButton 避免 Windows 按钮内边距撑大

### 侧边栏尺寸问题
- `setFixedWidth(50)` 显示为 60px
- 尝试 `Qt::AA_DisableHighDpiScaling` → Qt6 已废弃，无效
- QPushButton 的 Windows 样式内边距撑大了侧边栏
- **最终修复**：QLabel 替换 QPushButton + `qputenv("QT_ENABLE_HIGHDPI_SCALING", "0")`

### 关键文件
- `src/app/ToolboxWindow.h / .cpp` — 主窗口布局
- `src/main.cpp` — 入口 + DPI 禁用
- `src/file-manager/` — 文件管理器模块（未修改）
